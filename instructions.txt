install babel and webpack: npm install -D babel-loader @babel/core @babel/preset-env webpack webpack-cli
to run auto jest: terminal => npm run watch


to test, use npx webpack serve

to deploy,
1. git branch gh-pages (only if branch never made)
2. git checkout gh-pages && git merge main --no-edit
3. npx webpack
4. git add dist -f && git commit -m "Deployment commit"
5. git subtree push --prefix dist origin gh-pages
6. git checkout main

7. change source branch in github to gh-pages
